CTSALib / CTSA

GachiFit uses CTSALib, a Swift Package wrapper for selected CTSA time-series analysis functions.

CTSALib includes and wraps portions of CTSA. Only selected CTSA functions are exposed through CTSALib; it is not a complete wrapper for all CTSA APIs.
